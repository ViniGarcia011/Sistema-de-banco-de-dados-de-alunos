# Sistema-de-banco-de-dados-de-alunos
É um sistema de um banco de dados de uma escola, que guarda nome do aluno, idade e nota. Consigo adicionar novo aluno, ver lista de alunos, deletar um aluno, buscar aluno pelo nome, e mostrar a media de notas de todos eles. Foi uma prova de um curso que estou fazendo de Automação em python, finalizei a parte básica com essa prova, agr a automação..
